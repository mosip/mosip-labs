# GH-Tracker
NOTE: Before installing the below Helm charts, make sure to update the required (mandatory) variables in the `gh-tracker-ui-values.yaml` and `gh-tracker-values.yaml` files.

## Install
```
./install.sh
```

## Restart
```
./restart.sh
```

## Delete
```sh
./delete.sh
```

## GH-Tracker UI portal
Access the portal with following URL:
```
https://<your-internal-api-host>/gh-tracker-ui/

Example:
https://api-internal.sandbox.xyz.net/gh-tracker-ui/
```
Your wireguard client must be running for this access.
